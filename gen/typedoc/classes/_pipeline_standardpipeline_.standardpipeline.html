Class StandardPipeline

Primary compilation pipeline. Takes one raw, uncompiled page from the pipeline input, and writes one compiled, pure-html page to the pipeline output. The produced page is fully compiled and ready to be rendered by a standard web browser. Additionally, it will be formatted by any HTML Formatter provided.

Any incidental resources (such as stylesheets) will be fed to the pipeline interface via createResource().
